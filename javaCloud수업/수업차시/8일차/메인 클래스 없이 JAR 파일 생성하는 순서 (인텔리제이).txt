메인 클래스 없이 JAR 파일 생성하는 순서 (인텔리제이)
1. 프로젝트 빌드 (클래스 파일 생성)
먼저 .java 소스 파일을 .class 파일로 변환해야 합니다.

인텔리제이에서 프로젝트를 빌드
Build → Build Project (Ctrl + F9) 실행
성공적으로 빌드되면, out/production/{프로젝트명} 폴더에 .class 파일이 생성됨
2. JAR 아티팩트 생성
이제 .class 파일들을 JAR 파일로 묶어야 합니다.

File → Project Structure (Ctrl + Alt + Shift + S) 클릭
좌측에서 Artifacts 선택
우측 상단 + 버튼 클릭 → JAR → Empty 선택
JAR 이름을 설정 (예: mylibrary)
3. 클래스 파일 추가 (빌드된 결과물 포함시키기)
Output Layout 탭으로 이동
+ 버튼 클릭 → Directory Content 선택
out/production/{프로젝트명} 폴더 선택 (이 폴더에 컴파일된 .class 파일이 있음)
Apply → OK 클릭
4. JAR 빌드 (JAR 파일 생성하기)
Build → Build Artifacts... 클릭
생성한 아티팩트 선택 → Build 클릭
out/artifacts/{아티팩트명}_jar/ 폴더에 .jar 파일이 생성됨
완료! 🎉
이제 메인 클래스 없이도 라이브러리 용도로 사용할 수 있는 JAR 파일이 만들어졌습니다. 🚀
